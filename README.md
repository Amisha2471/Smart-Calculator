# 🧮 Smart Calculator

A feature-rich calculator application built using Python that performs basic as well as advanced mathematical operations.  
This project demonstrates the use of Python functions, conditional statements, user input handling, and mathematical libraries.

## 🚀 Features

### Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Power (**)

### Advanced Operations
- Square Root
- Trigonometric Functions:
  - sin()
  - cos()
  - tan()
- Logarithmic calculations
- Percentage calculations

## 🛠️ Technologies Used

- Python
- Math Library
- Streamlit (if you used Streamlit)
